Split foyer

A VPN configuration that allows for selective routing of network traffic.

Description

Split foyer, often referred to as split tunneling in the context of VPNs, is a networking concept where a user can access both a public network (like the internet) and a private network (such as a corporate intranet) simultaneously. With this configuration, only specific traffic destined for the private network is routed through the VPN, while other traffic accesses the internet directly without the VPN. This can improve performance and reduce latency for non-sensitive activities while still providing a secure connection for corporate resources. For example, a remote worker can use a VPN to access company files and applications while still being able to browse the web without the overhead of the VPN. This configuration is particularly useful for organizations that want to maintain security for sensitive data while allowing users to take advantage of faster internet speeds for less critical tasks.

Examples

  • Accessing company intranet while browsing the internet directly.
  • Using cloud services for personal use without affecting VPN performance.

Additional Information

  • Split tunneling can pose security risks if not managed properly.
  • Some VPN providers may not support split tunneling.
