Blast

Blast is a secure remote access technology used in VPN environments to deliver applications and desktops to end users.

Description

Blast is a component of the VMware Horizon platform that provides a streamlined and secure method for delivering virtual applications and desktops to users over a VPN. It utilizes advanced protocols to optimize the user experience, ensuring that applications are responsive and efficient, even in less than ideal network conditions. Blast is designed to work seamlessly with various devices and operating systems, allowing users to access their work environments from anywhere, at any time, while maintaining a high level of security. The technology incorporates features such as adaptive bitrate streaming, which adjusts the quality of the transmitted data based on the available bandwidth, thus improving performance. Furthermore, Blast supports the use of secure connections through SSL/TLS, protecting sensitive data as it travels over the internet. Overall, Blast enhances productivity and flexibility for remote workers while ensuring corporate data remains secure.

Examples

  • Accessing a corporate application from a remote location using a personal device.
  • Utilizing virtual desktops for remote employees in a secure manner.

Additional Information

  • Blast is part of VMware's broader Horizon Cloud services.
  • It can function over various network protocols, including HTTP/HTTPS.
